## Project Purpose
This project was built to improve problem-solving skills and simulate real exam environments.


# 🌟 PrepWise
PrepWise is an interactive quiz-based web application designed to help students prepare for competitive exams in a structured and engaging way.
It allows users to attempt quizzes, navigate between questions, and track their progress efficiently.

##  Features

-  Dynamic quiz system
-  Question navigation panel
-  Answer tracking (attempted/unattempted)
-  Smooth user experience
-  Score calculation 
-  Timer support 

## Tech Stack
- HTML
- CSS
- JavaScript
